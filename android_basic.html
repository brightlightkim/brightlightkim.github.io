View Objects

Views are the building blocks to compose a UI
Everything on the screen is a view. Views that the user can see or interact with are called widgets.

Every widget is an instance of the View class or one of its subclasses (such as TextView or Button)


View Group

ViewGroup is a kind of View that contains and arranges other views.
ViewGroup does not display content itself. ViewGroups are often referred to as layouts.
In the Default activity layout, ConstraintLayout is the ViewGroup responsible for laying out its sole child, a TextView widget
LinearLayout inherits from ViewGroup. Other ones include FrameLayout as well.

  • android:layout_width >> for width
  • android:layout_height >> for height
  • match_parent >> big as parent
  • wrap_content >> big as its content require
  • andoird:orientation >> can determine if it is vertical or horizontal
  • andoird:text >> what text to display. it can reference such as @string/syntax
