What is Fragment?

Fragment is a controller object that an activity can deputize to perform tasks.
Most commonly the task is managing a UI
UI can be an entire screen or just one part of the screen
Fragment managing a UI is known as a UI fragment.
Activity's view can hold a container for the fragment.
The fragment's view is inserted into the container once it is inflated.
An activity can have multiple containers in its view for different fragments
